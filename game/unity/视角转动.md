## 物体按某个轴心转动

```c#
    Transform barrel;
    float min = -30;
    float max = 30;
    float elevation =0;

    void Update(){
        transform.Rotate(0,Input.GetAxise("Horizontal"),0);
        
        float v = Input.GetAxis('Vertical');
        elevation = Mathf.Clamp(elevation+v,min,max);
        barrel.localRotation = Quaternion.Euler(elevation,0,0);
    }
```
